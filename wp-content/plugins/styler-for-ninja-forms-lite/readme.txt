Style Ninja Forms with no css knowledge

== Description ==
Styler for Ninja Forms is for users who want to style Ninja Forms without any CSS knowledge. It is built on WordPress Customizer that shows you live preview of design changes.

You can also preview you form design for different devices like desktops, tablets, mobiles directly from customizer and give different font size and field widths for them.

Design settings are divided into multiple sections for easy access. You can design Form Wrapper, Form Title, Labels, Description, Text Fields, Dropdowns, Radio, Checkboxes, Paragraph Textareas,  Confirmation message, error message and submit button. 

There are controls to change field width, field height, font size, font weight, line height, text decoration, change font to upper case, make font italics, background color, border color, border size, border style, border radius, font style, font alignment, margin, padding, etc

You can design unlimited forms and make unlimited design revisions. There are no limits on how you use this addon.

You can give can use units like px, %, em, rem, etc for the font size, width, height. For adding color you can either enter the values in hex code s or use the color picker to select the color of your choice.


== Installation ==
1. Upload the `ninja-forms-styler-lite` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to page where you have form in frontend.
4. Click on customize in top bar.
